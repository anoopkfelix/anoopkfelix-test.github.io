To know git version

git --version

To set git global username and user email

git config --global user.name "anoopkfelix"

git config --global user.email

To set the username/e-mail for the current repo, remove global

git config user.name "anoopkfelix"

git config user.email "antonyanoop@live.com"

To make new directory

mkdir new_folder

To open any directory

cd new_folder

To initiate git

git init

To list the files in git

ls

To add a newly created or edited file to staging

git add index.html

To add all files in the directory

git add --all

git add -A

To know th status of the files in the current directory

git status

or to know short status

git status --short

To commit a file

git commit -m "First release of Hello World!"

To commit directly without stage

git commit -a -m "Updated index.html with a new line"

To view log

git log

For help

git command -help

git help --all

to reach end of the help list

SHIFT + G

The press q to quit from the list

q

To create a new git branch

git branch hello-world-images

To confirm new branch created or not ?

git branch

A branch is a new/seperate version of teh main repository.

To create a new branch

git branch hello-world-images

To confirm that we have created a new branch

git branch

checkout is the command used to move from the current branch, to the one specified at the end of the command:

git checkout hello-world-images

Note: Using the -b option on checkout will create a new branch, and move toit, if it does not exist.

git checkout -b new-world
